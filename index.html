Proyecto en Git

Demostración del uso de Git y sus funcionalidades.

¿Qué es Git?

Git es un sistema de control de versiones distribuido, utilizado principalmente para gestionar proyectos de desarrollo de software.

Características de Git

  • Seguimiento de cambios
  • Colaboración en equipo
  • Ramas y fusiones

¿Por qué usar Git?

  • Distribuido: Cada usuario tiene una copia completa del repositorio.
  • Rendimiento: Optimizado para manejar proyectos grandes.
  • Seguro: Garantiza la integridad de los datos con criptografía.

Comandos Básicos de Git


git init

Inicializa un nuevo repositorio en el directorio actual.

git add

Añade archivos al área de preparación (staging).

git commit

Confirma los cambios añadidos al repositorio local.
